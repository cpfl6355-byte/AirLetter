# AirLetter 모바일 4탭 내비게이션 및 유리 질감 UI 개편 완료

사용자님께서 요청하신 **4가지 카테고리 중심의 고기능성 3D 곰돌이 미세먼지 서비스**로 UI/UX 및 라우팅 구조를 전면 리뉴얼하였습니다. 

---

## 1. 🌟 주요 개편 사항

### 📊 탭 1: 지역현황 (실시간 수치)
- **메인 기능:** fixed 배경 위에 투명한 유리 질감 카드가 얹어진 홈 화면입니다. 
- **곰돌이 3D 마스코트:** 대기질 상태(좋음, 보통, 나쁨, 매우나쁨)에 맞춰 마스코트가 상단에 플로팅 애니메이션과 함께 등장합니다.
- **연동 정보:** 실시간 미세먼지 수치(PM10, PM2.5)와 나의 프로필 기반 **건강 위험도 지수**가 요약 표시됩니다.

### 🛡️ 탭 2: 행동가이드 (맞춤 요령)
- **독립 탭 구성:** 홈 화면에 길게 늘어져 있어 보기 힘들었던 가이드 영역을 단독 탭으로 격상시켰습니다.
- **수치 기준 & 맞춤 행동:** 상단 탭을 눌러 각 등급별(좋음/보통/나쁨/매우나쁨) 곰돌이 비주얼과 구체적인 대기 수치 범위, 맞춤형 예방 가이드 행동 카드를 쉽고 명확하게 확인할 수 있습니다.

### 👥 탭 3: 알림·보호 (가족 연동)
- **가족 안심 모니터링:** 가족 멤버들의 거주 지역 미세먼지 수치 및 대기질 등급을 한눈에 조회합니다.
- **가족 추가 폼:** 보호할 가족의 호칭, 거주 지역, 건강 특징(호흡기 질환, 노약자, 어린이 등)을 선택해 간편히 등록할 수 있습니다.
- **알림 설정:** 미세먼지 경보 및 가족 안심 경보 토글 설정 카드를 이 화면에 함께 배치하여 직관성을 높였습니다.

### ❤️ 탭 4: 내 건강 (프로필·증상 기록)
- **종합 헬스케어 화면:** 기존에 쪼개져 있던 '프로필 설정', '증상 기록', '과거 기록 리스트', '데이터 초기화' 기능을 하나의 종합 대시보드로 정돈하여 매끄럽게 통합했습니다.
- **프로필 설정:** 나이, 지역, 민감 질환 종류를 체크하고 즉시 저장합니다.
- **증상 기록:** 오늘의 호흡기 및 신체 증상(기침, 콧물, 눈충혈, 두통 등)과 심각도(1~5단계)를 입력하여 나만의 건강 히스토리를 누적 관리할 수 있습니다.

## 3. 🧪 자가 테스트 가이드

터미널 보안 및 권한 이슈로 임시 로컬 서버 구동이 제한될 수 있으므로, 아래 방법으로 즉시 동작을 확인하실 수 있습니다:

1. **Vite 개발 서버 구동 (권장):**
   프로젝트 폴더 내 터미널에서 아래 명령어를 입력하여 로컬 데브 서버를 기동합니다.
   ```bash
   npm run dev
   ```
   이후 브라우저에서 표시되는 로컬 서버 주소(예: `http://localhost:5173`)로 접속합니다.

2. **화면 연동 확인:**
   - 하단 내비게이션 바의 **행동가이드, 알림·보호, 내 건강** 메뉴를 터치하며 자연스러운 Glassmorphism 스크린 슬라이드와 곰돌이 로딩 효과를 느껴보세요.
   - **가족 등록하기** 폼에 "아빠 (인천 · 노약자)"를 추가한 뒤 상단 모니터링 카드에 실시간 등급과 함께 카드 형태로 즉시 누적 렌더링되는 것을 확인하실 수 있습니다.
